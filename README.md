# Cybersecurity Internship Tasks

This repository contains cybersecurity tools developed using Python as part of internship tasks.

## Task 1 – File Integrity Checker

Detects file modifications by generating and comparing SHA-256 hash values.

## Task 2 – Web Vulnerability Scanner

Scans websites for SQL Injection and Cross-Site Scripting (XSS) vulnerabilities.

## Task 3 – Penetration Testing Toolkit

Performs host discovery, port scanning, and banner grabbing using socket programming.

## Task 4 – Encryption Tool

Encrypts and decrypts messages using Fernet symmetric key encryption.

## Technologies Used

* Python
* Socket Programming
* Requests Library
* Cryptography Library
